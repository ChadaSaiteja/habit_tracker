# Habit Tracker Lite

A small static habit tracker built with plain HTML, CSS, and JavaScript.

## What it does

- Tracks recurring habits month by month
- Lets you edit only today
- Shows bar, line, and streak views
- Stores data in browser local storage
- Works well on mobile and desktop

## Run locally

Open `index.html` in a browser, or serve the folder with any simple static server.

## Deploy

This repo can be deployed on any static host, including:

- GitHub Pages
- Netlify
- Vercel
- Cloudflare Pages

## Important note

This version stores data only in the browser on each device. If you open it on another phone or laptop, the data will not sync automatically.
